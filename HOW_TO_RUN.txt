MalCapture Defender detects malicious screen capture activities by:

1. PROCESS MONITORING
   - Detects suspicious screenshot utilities (psr.exe, SnippingTool, etc.)
   - Identifies hidden processes taking screenshots
   - Monitors processes in suspicious locations (TEMP, AppData)

2. API MONITORING
   - Tracks Windows GDI/GDI+ API calls (BitBlt, GetDC, etc.)
   - Detects rapid API call patterns
   - Identifies classic screenshot sequences

3. FILE SYSTEM MONITORING
   - Watches for rapid image file creation
   - Monitors suspicious directories
   - Detects sequential screenshot naming patterns

4. NETWORK MONITORING
   - Identifies screenshot exfiltration attempts
   - Monitors suspicious ports and connections
   - Tracks large image file transfers
